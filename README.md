概况
=
本实验使用三明治开发套件制作一款智能宠物喂食器，实现的功能如下所示：
* 定时喂食，周定时循环，食量可选
* 手机可操控，点击选择喂食份量
* 可以看喂食记录<br>

物料清单
=
* 涂鸦三明治直流供电电源板
* 涂鸦三明治 Wi-Fi MCU 通信板（WB3S）
* 减速电机
* NUCLEO-G071RB
